# The Memo Board

A mini project developed for BCG-DV as part of a Front End Developer interview challenge.

## Main Commands

#### Install Dependencies

```
npm install
```

#### Run Unit Tests

```
npm test
```

#### Start The Dev Server

```
npm start
```
